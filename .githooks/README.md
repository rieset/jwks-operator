# Git Hooks

Этот каталог содержит git hooks для проекта JWKS Operator.

## Установка pre-commit hook

Для установки pre-commit hook выполните:

```bash
git config core.hooksPath .githooks
```

Или создайте символическую ссылку:

```bash
ln -sf ../../.githooks/pre-commit .git/hooks/pre-commit
```

## Что делает pre-commit hook

Pre-commit hook автоматически выполняет следующие проверки перед коммитом:

1. **Проверка форматирования** (`go fmt`) - проверяет, что код правильно отформатирован
2. **Проверка импортов** (`goimports`) - проверяет правильность импортов
3. **Линтинг** (`golangci-lint`) - запускает полный линтинг кода

Если любая из проверок не проходит, коммит будет отклонен.

## Ручной запуск проверок

Вы можете запустить проверки вручную:

```bash
# Проверка форматирования
make fmt-check

# Линтинг
make lint

# Все проверки
make check
```

## Пропуск hook (не рекомендуется)

Если вам нужно пропустить hook (например, для экстренного коммита), используйте:

```bash
git commit --no-verify
```

**Внимание:** Не используйте `--no-verify` без необходимости, так как это может привести к коммиту некорректного кода.
